# Developer Portfolio

This is my personal developer portfolio website built with **React**.  
It showcases my projects, technical skills, experience, and professional journey as a software developer.

## Features

- Modern and clean UI
- Fully responsive design
- Projects showcase section
- Experience and education sections
- Skills and technologies overview

## Tech Stack

- React
- JavaScript
- Tailwind CSS
- Framer Motion
- Vite 

## Getting Started


```bash
git clone https://github.com/theahmednagra/personal-portfolio.git
cd personal-portfolio
npm install
npm run dev
